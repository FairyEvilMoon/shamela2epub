# Contributing

Contributions are welcome.

Please keep the project focused on exporting books from a user's own local Shamela 4 installation. Do not commit copyrighted book data, Shamela databases, generated EPUBs, or local output folders.

Useful areas for contribution:

- More robust cover extraction from `cover.db`.
- Better handling of footnotes and special inline markup.
- EPUB validation improvements.
- Tests using small synthetic SQLite/Lucene fixtures.
- Packaging for Windows/macOS/Linux.
